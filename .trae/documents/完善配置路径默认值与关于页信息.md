## 目标
- 配置保存路径缺失时填充默认值（userData）。
- 关于页 GitHub 链接改为官方仓库地址。
- 关于页简介从 README 读取并显示，失败时回退到内置描述。

## 实现步骤
1. 主进程 IPC
- 新增 `system.getUserDataDir`：返回 `app.getPath('userData')`。
- 新增 `system.getReadmeSummary`：读取项目根 `README.md`，提取标题与简介文本，返回字符串；失败时返回空。

2. 预加载层
- 暴露 `system.getUserDataDir` 与 `system.getReadmeSummary` 到渲染层。
- 更新 `global.d.ts` 类型声明。

3. 渲染层 Settings.vue
- `repoUrl` 默认改为 `https://github.com/ACFUN-FOSS/acfun-live-toolbox-MKII`。
- 在 `loadSettings()`：若 `config.dir` 缺失，调用 `system.getUserDataDir` 填充，并写入配置。
- 关于页新增简介文本 `toolIntro`；在 mounted 时调用 `system.getReadmeSummary` 设置，失败时使用内置描述作为回退。

4. 验证
- 运行类型检查，确保主/预加载/渲染三侧类型一致。
- 保持不启动开发服务器，不新增测试。
